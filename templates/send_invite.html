⚙️ Gmail Email Setup

To auto-send emails, set up Gmail App Password:

  1. Go to myaccount.google.com → Security → Enable 2-Step Verification

  2. Search "App Passwords" in Google account settings

  3. Click Create, name it "DocTracker" → copy the 16-character password

  4. Add to Railway → Variables:

    MAIL_SENDER = your@gmail.com
    MAIL_PASSWORD = xxxx xxxx xxxx xxxx
    APP_URL = https://docktracker.up.railway.app
